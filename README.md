# Тестовое задание VK: Автоматизация виджета (Python + Playwright)

Репозиторий содержит решение тестового задания на позицию тестировщика-автоматизатора. Исходный код был перенесен с TypeScript на Python с исправлением логических ошибок и соблюдением паттерна Page Object Model.

## 🛠 Что было исправлено
1.  **Логика ожидания (Async/Await):** В исходном коде отсутствовал `await` при клике на статью, что приводило к гонке условий (race condition). В Python-версии использованы встроенные механизмы ожидания Playwright.
2.  **Селекторы:** Оптимизированы селекторы в `WidgetPage` для повышения стабильности тестов (использованы более точные пути к кнопкам и элементам списка).
3.  **Инкапсуляция:** Локаторы вынесены в приватные поля класса (согласно PEP 8), а взаимодействие с ними происходит через методы страницы.
4.  **Обработка Cookie:** Добавлена проверка на наличие поп-апа с куками, который может перекрывать виджет.

## 🚀 Стек технологий
*   **Python 3.10+**
*   **Pytest** (тестовый фреймворк)
*   **Playwright** (библиотека автоматизации)

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/selenadanilova-ux/Uchi-widget-automation-python
   cd Uchi-widget-automation-python

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # для macOS/Linux
   # или
   venv\Scripts\activate     # для Windows

3. **Установите зависимости:**
   ```bash
   pip install pytest-playwright
   playwright install chromium

4. **Запустите тесты:**
   ```bash
   # Запуск всех тестов в фоновом режиме
   pytest

   # Запуск с отображением браузера (headed mode)
   pytest --headed

## 📂 Структура проекта

*   **pages/** — содержит `widget_page.py` (описание элементов виджета и методов взаимодействия с ними).
*   **tests/** — содержит `test_widget.py` тестовые сценарии (исправленные тесты и новые проверки).
*   `.gitignore` — исключает из репозитория системный мусор и виртуальное окружение.
*   `requirements.txt` — список необходимых библиотек для быстрого запуска.
